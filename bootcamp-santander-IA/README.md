# 🚀 Plano de Estudos: 30 Dias para Back-end Java

Bem-vindo ao seu plano de estudos focado em **Java, Banco de Dados, Git/GitHub e Spring Boot**. O objetivo principal é construir uma base sólida para desenvolver APIs REST e iniciar a construção do seu portfólio para vagas de estágio ou de desenvolvedor Júnior.

---

## 🎯 Objetivos do Plano

Ao final dos 30 dias, você será capaz de:
* ☕ **Entender profundamente** os fundamentos da Programação Orientada a Objetos (POO) em Java.
* 💾 **Manipular bancos de dados** utilizando PostgreSQL e consultas SQL.
* 🔧 **Utilizar Git e GitHub** para controle de versão e publicação de portfólio.
* ⚡ **Desenvolver APIs REST** completas utilizando Spring Boot.
* 📂 **Publicar seu primeiro projeto completo** no GitHub e planejar próximos passos.

---

## ⏰ Método de Estudo (1 hora por dia)

Para otimizar o aprendizado e manter a constância sem sobrecarregar sua rotina, utilize a seguinte divisão de tempo diária:

```mermaid
gantt
    title Divisão Diária (60 minutos)
    dateFormat  m
    axisFormat %M
    section Estudo
    Revisão (Conteúdo Anterior)  :active, 0, 10m
    Conceito Novo               : 10m, 35m
    Prática & Código            : 35m, 55m
    Anotações & Registro        : 55m, 60m
```

| Tempo | Atividade | Descrição |
| :--- | :--- | :--- |
| **10 min** | **Revisão** | Relembre rapidamente o que foi estudado no dia anterior para fixar o conteúdo. |
| **25 min** | **Conceito Novo** | Foque na teoria, assista a aulas ou leia a documentação recomendada. |
| **20 min** | **Prática** | Escreva código, faça exercícios e teste os conceitos na sua máquina. |
| **5 min** | **Anotações** | Registre os aprendizados, dificuldades e avanços no seu diário de estudos. |
